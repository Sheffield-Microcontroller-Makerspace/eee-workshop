# CS EEE Microcontroller Workshop

Welcome to the CS-EEE Microcontroller Workshop! Here you can find self-guided exercises along with example code for each exercise. It is **strongly** suggested you start with [exercise 0](exercises/00-hello-world.md) as it will not only get you introduced to the board and IDE, but will help us debug any issue that you encounter.
