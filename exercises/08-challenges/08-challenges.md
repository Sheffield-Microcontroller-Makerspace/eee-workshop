# Extension Challenges

## 1. Reaction-time Game (MEDIUM)

**Hardware:** button and LEDs.

**Requirements:**
- Wait for a random amount of time
- Turn on the LED
- Measure how long the player takes to press the button
- Print the reaction time to Serial
- Detect and penalise early presses

**Possible Extension:**
- Webpage display

## 2. Potentiometer Lock (HARD)

**Hardware:** button, potentiometer, and LEDs

**Requirements:**
- Use the potentiometer as a dial
- Use the button to submit a number
- Require a sequence of positions
- Green light means unlocked
- Red light means incorrect

**Possible Extensions:**
- Limited number of attempts
- Option to reset attempt

## 3. Light Synthesiser (MEDIUM)

**Hardware:** Extra breadboard, nine LEDs, potentiometer, and button

**Requirements:**
- Potentiometer can control speed
- Button switches patterns
- Include at least three patterns

**Pattern Recommendations:**
- Wave
- Random Sparkle
- Pulses

## 4. Binary Counter (MEDIUM)

**Hardware:** LEDs and button

**Requirements:**
- Press the button increment the counter
- Display the number in binary using the LEDs
- Print the decimal and binary values to Serial

Note: If you're stuck look up a ripple counter

## 5. Memory Game (HARD)

**Hardware:** multiple LEDs and buttons

**Requirements:**
- Generate a sequence
- Player reproduces it using buttons
- Sequence gets longer after each success
- Red LED indicates failure

Note, if you only have one or two buttons, you can use a potentiometer to set the colour and a button to confirm it.

## 6. Morse Code Transmitter (VERY HARD)

**Hardware:** Button and LED

**Requirements:**
- Count button presses and translate into morse code
- Display messages on webpage
